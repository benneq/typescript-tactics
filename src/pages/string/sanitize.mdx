### Convert to ASCII

```ts showLineNumbers
string
  .normalize('NFKD') // normalize accents and diacritics
  .replaceAll(/[^\x00-\x7F]+/g, ''); // remove remaining non-ascii characters
```

### Only Letters (from every language)

```ts showLineNumbers
string.replace(/[^\p{L}]/gu, '');
```

### Only Visible Characters and Space

```ts showLineNumbers
string.replace(/[^\p{C} ]/gu, '');
```
